# 🌗 Theme Toggle Project

This is a simple and elegant **Dark/Light Mode Toggle** website built using HTML, CSS, and JavaScript. It demonstrates how to manipulate the DOM, apply class toggling, and store user preferences using `localStorage`.

---

## 📸 Screenshot

![Theme Toggle Preview](https://via.placeholder.com/800x400.png?text=Theme+Toggle+Project+Preview)

---

## 💡 Features

- 🌞 Light & 🌚 Dark mode toggle
- ✅ User preference saved with `localStorage`
- 🎯 Smooth transitions with CSS
- 📱 Responsive layout

---

## 🚀 How It Works

1. Click the toggle button to switch between themes.
2. Your selected theme is saved in the browser using `localStorage`.
3. When you refresh or revisit the page, your preference is restored automatically!

---

## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- DOM Manipulation
- `localStorage`
